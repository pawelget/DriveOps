## Jak uruchomić
1. Sklonuj repo:
   git clone https://github.com/pawelget/DriveOps

2. Wejdź do katalogu:
   cd DriveOps

3. Uruchom kontener:
   docker compose up -d

## Dane logowania

- password: admin
- database: CarManagementDB
- port: 5432
